USM V7 - Research Uniplexed Souza Manske Version 7 Operating System

An operating system based on UNIX™ 7th Edition, being refactored and modernized with support for x86 (x86_32, x86_64), modern CPU vulnerability mitigations (e.g., Meltdown), optimized memory paging, an advanced scheduler, and modern file systems. It will include multiple ABI support for Linux and FreeBSD compatibility (per-process syscall tables) and future support for kernel modules.
